# Palanca

Send heartfelt goodbye palanca letters to PSHS-CVisC Batch 2022 students!

🏷️ Status: Active \
🤖 Vibe Coded: No

---

## What it does

- Receive letters from everyone in Pisay.
- Send with a name or anonymously.
- Check out all of the letters you sent to other people.
